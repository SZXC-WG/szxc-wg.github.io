## 첫 출발은 GitHub Releases

지금 바로 LocalGen을 플레이하고 싶다면 공식 GitHub Releases가 가장 안전한 출발점입니다.

- [LocalGen GitHub Releases 열기](https://github.com/SZXC-WG/LocalGen-new/releases)

이 사이트의 [릴리스 페이지]({{< relref "releases" >}}) 에서는 공개 이력과 배포 흐름을 비교하며 볼 수 있습니다.

## 어떤 라인을 선택해야 할까

### 바로 플레이하고 싶은 사람

**최신 안정판** 부터 시작하는 것이 가장 무난합니다. 다운로드 후 첫 경기까지 가장 부드럽게 이어집니다.

### 새 기능을 먼저 보고 싶은 사람

**미리보기 빌드** 와 개발용 빌드는 최신 Bot, UI, 도구 체인의 변화를 먼저 확인하고 싶은 사람에게 적합합니다.

## 기억해 둘 점

- **`master` / `v6.x`** 는 미래를 이끄는 Qt 라인입니다
- **v5.x** 는 기존 빌드와 장기 유지보수를 책임집니다
- 오래된 릴리스는 폰트나 실행 환경에 추가 주의가 필요할 수 있습니다
