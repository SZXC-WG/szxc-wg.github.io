## LocalGen が魅力的な理由

**LocalGen** は、すぐ遊べるローカル戦略ゲームであると同時に、Bot 研究や開発参加まで視野に入れたオープンソースの実験場です。

README が強調しているのは、次の三つです。

- generals.io 風の対戦を **完全オフライン** で遊べること
- **内蔵 Bot** ですぐに対戦や検証を始められること
- **同一 LAN** の友人とローカル対戦できること

## 知っておきたいブランチ

1. **`master` / `v6.x`**
   - Qt ベースの新しい主流ライン
   - UI、アーキテクチャ、Bot インターフェースの刷新が進行中
   - テスターやコントリビューター向けの最前線

2. **`v5.x`**
   - 既存ビルドと長期保守を支えるライン
   - 歴史的な配布物を追いたい人にも重要

> まず安心して遊び始めたいなら、最新の master をそのまま使うより、公開済みのリリースから入る方が安全です。

## 技術の見どころ

- **C++17** を中心とした実装
- **Qt6** と **CMake** による近代化された v6 系統
- リプレイ、カスタムマップ、Bot、CLI シミュレーターを含む広い実験余地
- GitHub ベースのオープンな議論とコラボレーション
