Esta sección refleja el historial público de lanzamientos de LocalGen y enlaza cada entrada con GitHub, donde viven los archivos descargables canónicos.

Úsala cuando quieras elegir un build seguro con confianza, seguir cómo evolucionó el proyecto o detectar ramas de vista previa que merecen probarse antes de convertirse en el siguiente hito estable.

De forma local, este sitio usa datos públicos sembrados. En GitHub Actions, el flujo de sincronización actualiza los datos con solicitudes autenticadas para mantener la cronología al día.
