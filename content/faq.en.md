## Is LocalGen affiliated with generals.io?

No. LocalGen is an independent, fan-made, open-source project. It is **not affiliated with, endorsed by, or sponsored by** generals.io.

## Which branch represents the newest work?

The newest active development line is **`master` / `v6.x`**, which is the Qt-based rewrite.

## Are older releases still important?

Yes. The project’s release history matters because:

- many users still want stable historical builds
- older releases document gameplay evolution clearly
- the older v5-era line remains relevant for long-term maintenance and legacy behavior

## Can I write my own bot?

Yes. The project supports built-in bots and is moving toward broader support for external bots through a network interface.

## Where should I report bugs or propose features?
