## Resumen de licencias del sitio

El repositorio `SZXC-WG/szxc-wg.github.io` es la fuente de este sitio web.

| Área | Estado |
| --- | --- |
| Diseños, estilos, JavaScript y visuales SVG generados | GPL-3.0 salvo que se indique lo contrario |
| Metadatos sincronizados del proyecto y datos de releases | derivados de los materiales del proyecto upstream LocalGen |
| Nombres y marcas de terceros como `generals.io` | pertenecen a sus respectivos titulares |

## Qué significa esto en la práctica

- salvo que un archivo indique otra cosa, el código del sitio y los SVG generados siguen la licencia **GPL-3.0**
- el arte upstream de LocalGen usado por este sitio sigue atribuyéndose al proyecto LocalGen
- las referencias a `generals.io` son solo descriptivas y **no implican afiliación ni respaldo**

```text
Repositorio: SZXC-WG/szxc-wg.github.io
Proyecto fuente: SZXC-WG/LocalGen-new
Licencia del sitio web: GPL-3.0
```

Para los detalles de licencia del proyecto, compara también el [repositorio fuente](https://github.com/SZXC-WG/LocalGen-new) y la página principal [Acerca de]({{< relref "about" >}}).
