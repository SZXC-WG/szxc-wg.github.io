## 왜 LocalGen에 주목할 만한가

**LocalGen** 은 단순히 즐길 수 있는 로컬 전략 게임이 아니라, Bot 연구와 개발 참여까지 이어지는 오픈소스 실험장입니다.

README가 강조하는 핵심은 다음과 같습니다.

- generals.io 스타일 경기를 **완전 오프라인** 으로 플레이할 수 있음
- **내장 Bot** 으로 즉시 대전과 검증을 시작할 수 있음
- **같은 LAN** 의 친구와 로컬 멀티플레이를 즐길 수 있음

## 알아두면 좋은 브랜치

1. **`master` / `v6.x`**
   - Qt 기반의 새로운 주력 라인
   - UI, 아키텍처, Bot 인터페이스가 활발히 개선 중
   - 테스터와 기여자에게 가장 중요한 최신 흐름

2. **`v5.x`**
   - 기존 빌드와 장기 유지보수를 담당
   - 역사적인 배포물과 예전 워크플로를 확인하는 데도 중요

> 일단 안정적으로 플레이를 시작하고 싶다면, 최신 master를 바로 쓰기보다 공개된 릴리스부터 시작하는 편이 더 안전합니다.

## 기술적으로 흥미로운 점

- **C++17** 중심 구현
- **Qt6** 와 **CMake** 로 현대화 중인 v6 계열
- 리플레이, 커스텀 맵, Bot, CLI 시뮬레이터까지 포함한 넓은 실험 공간
- GitHub 기반의 개방적인 협업 구조
