## 最初の一歩は GitHub Releases

LocalGen を今すぐ遊びたいなら、まずは公式の GitHub Releases を見るのが最も安全です。

- [LocalGen の GitHub Releases を開く](https://github.com/SZXC-WG/LocalGen-new/releases)

このサイトの [リリースページ]({{< relref "releases" >}}) では、公開履歴と配布の流れを見比べられます。

## どの系統を選ぶべきか

### まずは遊びたい人

**最新の安定版** から始めるのがおすすめです。最短でプレイに入りやすく、初回体験も安定しています。

### 新機能を追いたい人

**プレビュー版** や **開発中ビルド** は、最新の Bot、UI、ツールチェーンの変化を先に試したい人向けです。

## 覚えておきたいこと

- **`master` / `v6.x`** は将来を担う Qt 系統です
- **v5.x** は既存ビルドと長期保守を支えます
- 古いリリースではフォントや実行環境に追加注意が必要なことがあります
