## 如何参与贡献

LocalGen 明确欢迎社区通过 GitHub issue 和 pull request 参与项目建设。

## 主要贡献路径

- 在活跃的 `master` / `v6.x` 主线上进行 **玩法与引擎开发**
- 通过 `src/bots/` 参与 **内置 Bot** 开发
- 基于网络接口探索 **外部 Bot**
- 完善 README 与指南等 **项目文档**
- 通过 issue 提交 **Bug 报告与功能建议**

## 社区协作期待

- 让改动保持聚焦、便于审查
- 说明你改了什么、为什么这样改
- 遵循项目的提交信息规范
- 对 Bot 或较大的玩法改动，尽量附带回放、测试或基准说明
