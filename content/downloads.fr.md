## Commencez par GitHub Releases

Si vous voulez jouer à LocalGen sans détour, la page GitHub Releases reste la source canonique et la plus sûre.

- [Ouvrir les releases LocalGen sur GitHub](https://github.com/SZXC-WG/LocalGen-new/releases)

La page [Versions]({{< relref "releases" >}}) de ce site vous aide à comparer les sorties avant de choisir un build.

## Quel type de version choisir ?

### Pour jouer immédiatement

Commencez par la **dernière version stable**. C’est la meilleure voie pour passer rapidement du téléchargement à votre première partie.

### Pour tester l’avenir du projet

Les **builds d’aperçu** et les builds de développement conviennent mieux si vous voulez essayer les derniers changements sur les bots, l’interface et les outils.

## À retenir

- **`master` / `v6.x`** porte la direction Qt du projet
- **v5.x** conserve les builds plus anciens et la maintenance longue durée
- certains anciens builds demandent plus d’attention côté polices ou environnement d’exécution
