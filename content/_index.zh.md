欢迎来到 **Local Generals.io（LocalGen）** 的源文档驱动型项目网站。该项目由 **SZXC Work Group** 维护，是一个面向本地离线环境的开源策略游戏项目。

上游 README 把项目定位说得很直接：你可以在 **完全离线** 的情况下体验 generals.io 风格玩法，使用 **开箱即用的内置 Bot**，也可以和局域网中的朋友直接对战。本网站则把这些信息整理成一个更容易查阅的双语入口，帮助你选择版本、理解 Bot 体系、阅读模拟器流程，并按项目原本的术语参与协作。

有两点尤其值得先记住：

- **`master` / `v6.x`** 是正在推进中的 Qt 重构主线，仍处于持续演进阶段。
- **GitHub Releases** 仍然是获取旧版稳定构建和历史版本的最可靠入口。

如果你想快速判断该下载哪个版本、Bot 贡献需要满足什么条件，或者需要和项目文档保持一致的表达方式，这个站点就是为这些场景准备的。
