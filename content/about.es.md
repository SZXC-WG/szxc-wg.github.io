## Por qué LocalGen merece atención

**LocalGen** no es solo un juego de estrategia local. También es un terreno de pruebas para bots y un proyecto open source al que sí puedes contribuir.

El README del proyecto deja tres promesas muy claras:

- jugar partidas estilo generals.io **completamente offline**
- lanzar **bots integrados listos para usar** sin configuración extra
- enfrentarte a amigos en la **misma LAN** cuando prefieras multijugador local

## Las ramas que conviene conocer

1. **`master` / `v6.x`**
   - la línea activa de desarrollo
   - la reescritura Qt en curso
   - el lugar donde se prueba y construye el futuro del proyecto

2. **`v5.x`**
   - la rama de mantenimiento a largo plazo
   - donde siguen disponibles builds antiguos y jugables
   - útil para ver el historial y el comportamiento heredado

> Si quieres empezar a jugar con tranquilidad, suele ser mejor comenzar por las releases publicadas antes que usar directamente la rama más reciente.

## Qué hay de interesante técnicamente

- implementación centrada en **C++17**
- transición v6 con **Qt6** y **CMake**
- margen amplio para replays, mapas personalizados, bots y simulación CLI
- colaboración abierta basada en GitHub
