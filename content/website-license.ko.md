## 사이트 라이선스 개요

이 웹사이트의 소스는 `SZXC-WG/szxc-wg.github.io` 저장소에서 관리됩니다.

| 대상 | 상태 |
| --- | --- |
| 레이아웃, CSS, JavaScript, 생성 SVG 비주얼 | 별도 표기가 없으면 GPL-3.0 |
| 동기화된 프로젝트 정보와 릴리스 데이터 | 상위 LocalGen 프로젝트에서 파생 |
| `generals.io` 등의 이름과 상표 | 각 권리자에게 귀속 |

## 실무적으로 이해하면

- 따로 명시되지 않았다면 이 사이트의 코드와 생성 비주얼은 **GPL-3.0** 을 따릅니다
- 상위 프로젝트에서 가져온 소재와 데이터는 LocalGen 프로젝트에 귀속됩니다
- `generals.io` 언급은 설명 목적이며, 제휴나 승인을 의미하지 않습니다
